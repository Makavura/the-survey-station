# Survey REST API
git clone git@github.com:Makavura/the-survey-station.git

## Set ENV Configs in .env
- JWT_SECRET
- MONGODB_URL

## Installation

- cd the-survey-station 
- npm install
- npm run start

# watch mode
- npm run start:dev

# production mode
- npm run start:prod
